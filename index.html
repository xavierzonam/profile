Jawaban :

Website adalah kumpulan dari file-file yang dapat diakses melalui internet dengan menggunakan sebuah alamat yang disebut domain. Domain adalah nama unik yang digunakan untuk mengidentifikasi sebuah website, misalnya bing.com. Domain harus terdaftar di sebuah penyedia layanan domain, yang akan menghubungkan domain dengan alamat IP dari server hosting. Server hosting adalah komputer yang terhubung ke internet dan menyimpan file-file website. Server hosting juga menjalankan software web server, yang bertugas untuk menerima permintaan dari browser pengguna dan mengirimkan file-file website yang diminta. Web server menggunakan protokol HTTP atau HTTPS untuk berkomunikasi dengan browser.

Hubungan antara web server, domain, dan hosting adalah sebagai berikut:

  • File website disimpan di server hosting
  • Pengunjung mengetikkan nama domain di browser
  • Browser akan meminta server untuk mengirim file website ke browser
  • Server mengirimkan file website di browser
  • Pengunjung dapat mengakses website
